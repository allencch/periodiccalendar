Що таке Періодичний Календар?

Періодичний Календар це програма, яка дозволяє:
  • відслідковувати менструальні цикли
  • прогнозувати періоди плідності, коли ймовірність завагітніти є найвищою
  • прогнозувати "безпечні" дні коли ймовірність завагітніти є найменшою
  • прогнозувати дату наступної менструації
  • занотовувати періоди вагітності
  • вираховувати номер тижня вагітності
  • прогнозувати дату пологів
  • вести особисті нотатки

Періодичний Календар дозволяє використовувати симпто-термальний метод, який дає найвищу точність під час прогнозування плідних днів. Користувач програми може вибрати спрощені варіанти цього методу, або взагалі використовувати календарний метод, який сам по собі є дуже ненадійним.

Автори Періодичного Календаря наполягають на необхідності консультацій із спеціалістами по природньому плануванню сім'ї. Ні зручний інтерфейс користувача, ні дана система допомоги не можуть замінити консультанта чи добру книжку по цій темі.
